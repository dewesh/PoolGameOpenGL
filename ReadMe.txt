Game Playing instruction
-------------------------------------------------------------------------
Keys
-------------------------------------------------------------------------
a -> anticlockwise rotate stick
d -> clockwise rotate stick
w -> increase power
s -> Decrease Power
Left Click -> Shoot
For Stick Angle Precision -> use mouse movement
